# Text-to-SQL Generator

Standardized implementation of Text-to-SQL Generator.

## Standard Project Structure

This project follows the repository-wide standard architecture:

```
57_Text_to_SQL_Generator/
│
├── dataset/
├── models/
├── resources/
├── src/
│   ├── processing/
│   ├── output/
│   └── app.py
│
├── requirements.txt
└── readme.md
```

## Setup & Running

1. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run app:**
   ```bash
   python src/app.py
   ```
