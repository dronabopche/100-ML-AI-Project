# Style Transfer Model

Standardized implementation of Style Transfer Model.

## Standard Project Structure

This project follows the repository-wide standard architecture:

```
38_Neural_Style_Transfer/
│
├── dataset/
├── models/
├── resources/
├── src/
│   ├── processing/
│   ├── output/
│   └── app.py
│
├── requirements.txt
└── readme.md
```

## Setup & Running

1. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run app:**
   ```bash
   python src/app.py
   ```
