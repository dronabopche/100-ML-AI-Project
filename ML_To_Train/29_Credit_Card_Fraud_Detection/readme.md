# Credit Card Fraud Detection

Standardized implementation of Credit Card Fraud Detection.

## Standard Project Structure

This project follows the repository-wide standard architecture:

```
29_Credit_Card_Fraud_Detection/
│
├── dataset/
├── models/
├── resources/
├── src/
│   ├── processing/
│   ├── output/
│   └── app.py
│
├── requirements.txt
└── readme.md
```

## Setup & Running

1. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run app:**
   ```bash
   python src/app.py
   ```
