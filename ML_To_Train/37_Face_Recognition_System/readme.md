# Face Recognition System

Standardized implementation of Face Recognition System.

## Standard Project Structure

This project follows the repository-wide standard architecture:

```
37_Face_Recognition_System/
│
├── dataset/
├── models/
├── resources/
├── src/
│   ├── processing/
│   ├── output/
│   └── app.py
│
├── requirements.txt
└── readme.md
```

## Setup & Running

1. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run app:**
   ```bash
   python src/app.py
   ```
