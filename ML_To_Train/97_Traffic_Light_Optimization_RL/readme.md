# Traffic Light Control Optimizer

Standardized implementation of Traffic Light Control Optimizer.

## Standard Project Structure

This project follows the repository-wide standard architecture:

```
97_Traffic_Light_Optimization_RL/
│
├── dataset/
├── models/
├── resources/
├── src/
│   ├── processing/
│   ├── output/
│   └── app.py
│
├── requirements.txt
└── readme.md
```

## Setup & Running

1. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run app:**
   ```bash
   python src/app.py
   ```
