# AI Web Scraping & Orchestrator Agent

Standardized implementation of AI Web Scraping & Orchestrator Agent.

## Standard Project Structure

This project follows the repository-wide standard architecture:

```
100_AI_Web_Orchestrator_Agent/
│
├── dataset/
├── models/
├── resources/
├── src/
│   ├── processing/
│   ├── output/
│   └── app.py
│
├── requirements.txt
└── readme.md
```

## Setup & Running

1. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run app:**
   ```bash
   python src/app.py
   ```
