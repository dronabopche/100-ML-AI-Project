# Action Recognition in Video

Standardized implementation of Action Recognition in Video.

## Standard Project Structure

This project follows the repository-wide standard architecture:

```
39_Video_Action_Recognition/
│
├── dataset/
├── models/
├── resources/
├── src/
│   ├── processing/
│   ├── output/
│   └── app.py
│
├── requirements.txt
└── readme.md
```

## Setup & Running

1. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run app:**
   ```bash
   python src/app.py
   ```
